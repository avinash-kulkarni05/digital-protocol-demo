### 2. STUDY PROCEDURES (study_procedures)

**procedure_item structure:**
```json
{
  "procedure_id": "PROC-001",
  "procedure_name": "Blood draw",
  "description": "Approximately 30mL of blood will be collected",
  "frequency": "At each study visit",
  "duration": "5-10 minutes",
  "is_optional": false
}
```

**Categorize procedures by phase:**
- screening_procedures: Tests to determine eligibility
- treatment_procedures: Procedures during active treatment
- follow_up_procedures: Post-treatment monitoring
- optional_procedures: Procedures requiring separate consent

**Search patterns:**
- "procedures", "tests", "examinations"
- "blood samples", "biopsies", "imaging"
- "physical examination", "ECG", "vital signs"

### 3. RISKS (risks)

**Severity categories:** mild, moderate, severe, life-threatening
**Frequency categories:** very_common (>10%), common (1-10%), uncommon (0.1-1%), rare (<0.1%), very_rare (<0.01%)

## SEARCH STRATEGY BY SECTION

| ICF Element | Typical Protocol Sections |
|-------------|--------------------------|
| Study Purpose | 1, 2, 3 (Synopsis, Introduction, Objectives) |
| Procedures | 7, 8 (Assessments, Schedule of Activities) |
| Risks | 6.5, 6.6, 13 (Safety, Toxicity Management) |
| Benefits | 1, 2 (Synopsis, Background) |
| Alternatives | 1, 2, 6 (Background, Treatment) |
| Compensation | 11, Protocol Appendix |
| Confidentiality | 11, 12 (Data Handling, Privacy) |
| Withdrawal | 5, 7 (Subject Discontinuation) |
| Genetic/Biobanking | 8, 10, Appendix (Exploratory, Samples) |
| Pregnancy | 5 (Eligibility), 6.5 (Safety) |
