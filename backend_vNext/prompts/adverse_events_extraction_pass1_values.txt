Extract ALL adverse event specifications from this clinical trial protocol PDF.

- AE/SAE definitions and criteria per ICH E2A
- DLT criteria (Phase 1/2 studies only)
- Severity grading systems (typically CTCAE)
- Causality assessment methods

## EXTRACTION RULES
---

1. **AE Definitions**: Look in "Adverse Events", "Safety Assessments" sections
2. **SAE Criteria**: Extract ALL 7 ICH E2A criteria if mentioned
3. **DLT Criteria**: Only for Phase 1/2 dose-escalation studies
4. Use CDISC controlled terminology codes where applicable
5. Number IDs sequentially: sae-001, aesi-001, dlt-001, etc.

Return ONLY valid JSON. No markdown fences, no explanations.
